快速入门

只需 3 分钟，在本地启动 Talon 多模态数据引擎并运行您的第一个智能体查询。

1

下载与运行

Talon 以单个二进制文件形式分发。下载适合您架构的版本并启动服务器。

Terminal Command
curl -fsSL https://talon.dev/install.sh | sh
Start Server
./talon-server --start
2

SQL 基础操作

Talon 完全兼容 PostgreSQL 协议。您可以创建表并插入数据，同时利用我们的 ACID 保证。

Create Table
CREATE TABLE users ( id UUID PRIMARY KEY, name TEXT, embedding VECTOR(1536) );
3

多模态尝试

这是 Talon 的魅力所在。在同一查询中混合使用 Key-Value 存储、向量搜索和关系型数据。

KV & Vector Hybrid
-- Set user session in KV SET session:user_123 = '{"context": "active"}'; -- Semantic Search with SQL filter SELECT * FROM users WHERE vector_similarity(embedding, [0.1, ...]) > 0.9 AND session:user_123->>'context' = 'active';
